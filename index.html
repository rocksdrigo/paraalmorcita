¡Bienvenida Schönecita!

Design by TEMPLATED arr. Rodrigo Escalona

Hace rato no te enviaba una página.

Del cielo bajó un pintor para pintar tu figura, pero no encontró color para tanta hermosura. Por eso eres blanquita.

No había hecho páginas porque estaba pensando en hacerlas útiles, en lugar de simplemente usar el conocimiento para mostrarte. Mucho interés en la utilidad, y obvio, aún hace falta mucho por aprender. ¡Pero hacer hacer esto es divertido!

Minetras más aprendo, el conocimiento se hace pesado y las tantas posibilidades cansan a la creatividad. Quizá por eso es tan extranordinaria la simplicidad. He pensado durante un tiempo que esa es la razón por la que no escribimos música, le tenemos tanto respeto a la creación y al conocimiento que tenemos, es mucho, es variado y entre tantas cosas no es fácil manejarse. Es por ello que cualquiera con una guitarra y amor se lanza a ese sueño de hacer música.

Y aquí voy yo con el sueño de hacer páginas web, con el ánimo de ser muy útil, usar este conocimiento para un día servir a la humanidad. Crear sitios bonitos y/o divertidos. Quizá algún día alguien necesite una página web para algo rápido, aunque ahora para investigar sobre alguien, uno visita perfiles de instagram. PERO, alguien que se respete mucho tiene una página web para contar su historia. Estaría genial hacer páginas personales para otros. Anyway. ¡Mira todo lo que he aprendido!

Esto es un template

He tomado un template nice para poder hacer las cosas más rápido. No lo hice antes porque no hubiera podido leer todo lo que tiene este template, está lleno de cosas, y por suerte, hay cosas que le faltaban que le agregué.
